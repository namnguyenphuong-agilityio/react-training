## Getting Started

- Clone repository from GitLab  

```git clone git@gitlab.asoft-python.com:nam.nguyenphuong/react-training.git```

- Checkout branch   
```git checkout -b feature/practice1```

- Change directory   
```cd practice1```

- Install dependencies   
```pnpm install```

- Start local server   
```pnpm run dev```
